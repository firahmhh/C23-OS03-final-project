## Seller

### Registrasi seller

- Endpoint
  - /seller/regist
- Method
  - POST
- Request Body
  - name = string
  - email = string
  - password = string , min 6
  - phone_number = string
  - location = string
  - roles = string
- Response

```json
{
  "message": "Input Success"
}
```

### Edit profile seller 
- Endpoint
  - /seller/profile/edit/:idSeller
- Method
  - PUT
- Request Params
  - id_seller = int
- Request Body
  - name = string
  - phone_number = string
  - location = string
  - roles = string
  - photo = file
  - nama_pemilik_rekening = string
  - nomor_rekening = string
  - nama_bank = string
- Response
```json
{
    "message": "Profile updated"
}
```
